El proyecto se desarrolló en JavaScript usando Node.Js.

Es una aplicación de consola con un menú interactivo, utilice el modulo de “colors” para mejorar la visualización de la aplicación y 
el modulo de “inquirer” para generar los menús de la aplicación.

Los datos persisten mediante un archivo con extensión JSON donde se guardan los datos que se van generando, para generar un id único para las tareas utilice el módulo de “uuid”.
En la aplicación puedes crear tareas, listar las tareas (se muestra la descripción de la tarea y si esta completada o no), listar tareas completadas (se muestra las descripción de la tarea y la fecha en que se completó), listar tareas pendientes (muestra las tareas que no han sido completadas), completar tareas (muestra todas las tareas y mediante la barra espaciadora puedes marcar o desmarcar las tareas para completar o dejarlas en pendientes), borrar tarea (muestra una lista de todas las tareas, al seleccionar una tarea la aplicación pide una confirmación antes de borrarla) y salir (esta opción permite al usuario finalizar la ejecución del programa).
